## ğŸš¨ Jika Ada Masalah

### **"Foto tidak muncul setelah upload"**
```
Solusi:
1. Refresh halaman (F5)
2. Kosongkan cache browser (Ctrl+Shift+Delete)
3. Coba upload lagi dengan foto berbeda
```

### **"Tombol tidak respond / hang"**
```
Solusi:
1. Tunggu 2-3 detik (mungkin sedang upload)
2. Jika tetap hang, refresh halaman
3. Pastikan koneksi internet stabil
```

### **"Pesan error 'File terlalu besar'"**
```
Solusi:
1. Gunakan foto dengan ukuran < 2 MB
2. Kompress foto terlebih dahulu
3. Coba gunakan format JPG (lebih kecil dari PNG)
```
